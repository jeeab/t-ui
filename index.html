📌 First things to know

  • The screen locks when it sleeps. The default unlock code is 1234 — change it in Settings → Lock PIN.
  • Taps landing in the wrong spot? Press Alt + C on the keyboard to re‑run screen calibration. Works even on the lock screen.
  • Double‑click the trackball to wake the screen and to go Home from anywhere.
  • The first start sits on the Meshtastic logo for 15–20 seconds — that's normal, don't unplug it.
What is T‑UI?

T‑UI is a friendly home‑screen “launcher” for the T‑Deck, built on top of the open‑source Meshtastic mesh‑radio firmware. Your T‑Deck still does everything Meshtastic does (off‑grid text messaging over long‑range radio, no cell service needed) — T‑UI just wraps it in a phone‑like grid of apps and adds a bunch of its own.

Everything is baked into the one file you flash above — all the apps below come with it.

How to use your T‑Deck

Getting around

  • Tap the touchscreen to open apps and press buttons.
  • Double‑click the trackball to go Home from anywhere, and to wake the screen.
  • Swipe the home screen sideways for a second page of apps. Long‑press a tile to rearrange.
  • Taps landing off target? Press Alt + C to re‑run screen calibration (works on the lock screen too).

The lock

  • The device locks itself when the screen sleeps, and asks for a code. The default code is 1234.
  • Set your own in Settings → Lock PIN → Change. Double‑click the trackball on the Home screen to lock it on purpose.

The apps

  • Mesh — the full Meshtastic messaging/nodes/map.
  • Maps — your own offline map with drop‑pins (see the Maps guide below).
  • Files — browse the SD card and internal memory; make folders, copy, trash.
  • Notes, Calculator, Stopwatch & Timer, Flashlight, plus games (Snake, Breakout) and a Metronome.
  • Share files (Wi‑Fi) — in Settings; sends files to/from your computer wirelessly (turn it on only when you need it).

Kill switch (privacy)

  • In Settings you can toggle the mesh radio off — it stops sending/receiving while the rest of the device keeps working.

Transferring files (the Share screen)

  • Open Settings → Share files → Open. It shows an address like ftp://192.168.1.42 and a login (tdeck / 1234).
  • On your computer, use a free FTP app like FileZilla (Host = that IP, Username tdeck, Password 1234, Port 21). Windows Explorer's built‑in FTP is unreliable — FileZilla works best. Keep the Share screen open while transferring.
Getting maps for your area

Map images aren't included (they'd be huge, and everyone wants their own region). You load maps for your area onto the microSD card yourself. T‑UI uses the same map format as Meshtastic, so any T‑Deck map guide works.

The short version

  • Maps are standard OpenStreetMap tiles — small .png squares organised by zoom level.
  • They go in a folder named /maps at the top level of the SD card, in /maps/<zoom>/<x>/<y>.png form.
  • Use a community “map tile downloader” for the Meshtastic T‑Deck to grab your area (pick a region and zoom range, it produces the /maps folder), then copy that folder onto the SD card.
  • Pop the card back in — the Maps app now shows your area, and you can long‑press to drop pins.

Tip: search “Meshtastic T‑Deck offline map tiles” for the current downloader tools and step‑by‑step. Start with a smaller zoom range so it doesn't take forever.

Make your own apps

You can write your own little apps and games for the T‑Deck in a simple language called Lua — just a text file, no programming tools needed. Even better: hand the app‑maker's guide to an AI assistant and describe the game you want.

  • An app is one file called main.lua, placed in /apps/<yourapp>/ on the SD card. It shows up as a tile automatically.
  • Apps draw text and boxes, play beeps, and respond to taps — full details (screen size, commands, a working example, and a copy‑paste prompt for an AI) are in the app‑maker's guide.
